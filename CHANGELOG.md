# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-05

### Added
- 🚀 Initial beta release
- 📱 WhatsApp integration via whatsapp-web.js
- 🤖 Kiro IDE integration via vscode-rest-control extension
- 🔌 MCP Server with tools: `send_whatsapp`, `send_whatsapp_media`, `get_gateway_status`
- 💓 Heartbeat scheduler for automated tasks
- 🎭 Interactive onboarding flow via WhatsApp chat
- 🌐 Multi-language support (English & Traditional Chinese)
- 📦 Cross-platform installer (install.js) for Windows/Mac/Linux
- 📦 Windows PowerShell installer (install.ps1)
- 🏗️ Layered architecture for future IDE support (Cursor, Windsurf, etc.)

### Features
- Gateway server on port 3000
- REST API for message handling
- Auto-forward WhatsApp messages to Kiro chat
- AI personality templates (steering files)
- Media file support (images, videos, documents)
