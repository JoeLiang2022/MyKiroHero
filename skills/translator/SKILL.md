# Translator Skill

This skill helps translate text between different languages.

## Capabilities

- Translate between any language pairs
- Detect source language automatically
- Provide multiple translation options when relevant
- Explain nuances and context

## How to Use

When the user asks for translation, identify the source and target languages, then provide an accurate translation.

### Example Queries

- "Translate 'hello' to Chinese"
- "翻譯這句話成英文"
- "What does 'ありがとう' mean?"

## Response Format

- Provide the translation clearly
- Include pronunciation hints for non-Latin scripts
- Explain any cultural context if relevant

## Supported Languages

- English ↔ Chinese (Traditional/Simplified)
- English ↔ Japanese
- English ↔ Korean
- And many more...

## Notes

- Default target language: Traditional Chinese (for Taiwan users)
- Preserve tone and style when possible
