## 技術問題
- Shell 完全卡住（echo 都 timeout），可能是之前的 node 下載腳本佔住了
- 網路下載也 timeout（Wikimedia 和 Unsplash 都連不上）
- 解法：不嵌入 base64，直接用外部 URL（需要瀏覽器開啟才能載入圖片）
- 教訓：**Wikimedia Commons > Unsplash** — 檔名就是描述，不會搞錯圖片內容

## 教訓
- **不要用隨機 Unsplash ID** — 無法從 ID 判斷圖片內容，容易放錯圖
- **Wikimedia Commons 檔名 = 內容描述** — 例如 `Fukuoka_Airport_International_Terminal_2.jpg` 一定是福岡機場
- **網路不穩時不要硬下載** — 改用外部連結，讓使用者的瀏覽器自己載入

### 過程

### 教訓
- **GitHub fine-grained token 權限要明確** — 建 repo 和寫入 repo 是不同權限
- **GitHub Contents API > git push** — 不需要 SSH/credential，直接 HTTPS API 上傳
- **先確認環境再選方案** — 沒有 SSH/gh CLI 就別走 git push 路線
