## 使用說明

在上方 ``` 區塊內加入排程，格式：`HH:MM 任務名稱`

**範例：**
```
09:00 早安提醒
04:00 記憶同步
```

Gateway 會自動讀取並在指定時間觸發任務。每小時重新載入，不需重啟。
