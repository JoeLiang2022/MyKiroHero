🤖 MyKiroHero — HLD

AI-Powered WhatsApp Gateway for Kiro IDE

📋 Overview

MyKiroHero 是一個將 Kiro IDE 的 AI 能力延伸到 WhatsApp 的 Gateway 系統。使用者透過 WhatsApp 發送訊息，Gateway 自動轉發給 Kiro AI 處理，再將回覆送回 WhatsApp。

Node.js Express MCP Protocol WhatsApp Web.js WebSocket

📦 Component Details

⚙️ Gateway Server

Express-based HTTP server，負責所有訊息路由與 API 端點。

  • REST API for message relay
  • Health check endpoint
  • Media file forwarding
  • Auto port detection

src/gateway/server.js

📱 WhatsApp Adapter

使用 whatsapp-web.js 連接 WhatsApp，處理 QR 登入與訊息收發。

  • QR code authentication
  • Message send/receive
  • Media support (images, videos, docs)
  • Session persistence

src/gateway/whatsapp-adapter.js

🔧 MCP Server

Model Context Protocol server，提供 AI 可呼叫的工具。

  • send_whatsapp — 發送文字訊息
  • send_whatsapp_media — 發送媒體檔案
  • get_gateway_status — 健康檢查
  • get_weather — 天氣查詢
  • list_skills / load_skill — 技能管理
  • restart_gateway — 重啟服務

src/mcp-server.js

💓 Heartbeat Scheduler

讀取 HEARTBEAT.md 排程設定，定時觸發任務。

  • Cron-like scheduling via markdown
  • Auto-reload every hour
  • Memory sync, reminders, weather

HEARTBEAT.md

🔷 IDE Handler

透過 vscode-rest-control extension 與 Kiro IDE 通訊。

  • HTTP POST to inject chat messages
  • Port auto-discovery (workspace hash)
  • Pluggable handler architecture
  • Future: Cursor/Windsurf support

src/gateway/handlers/kiro-handler.js

🎭 Steering Files

Markdown-based AI personality & memory system。

  • SOUL.md — AI identity & personality
  • USER.md — User preferences
  • MEMORY.md — Long-term memory
  • AGENTS.md — Behavior rules
  • HEARTBEAT.md — Scheduled tasks

.kiro/steering/

🧱 Layered Architecture

graph LR subgraph Universal["🌐 Universal Layer"] G[Gateway Server] API[REST API] WA2[WhatsApp Adapter] MCP2[MCP Server] HND[Handler Abstraction] end subgraph KiroLayer["🔷 Kiro-Specific Layer"] KH[kiro-handler.js] ST[Steering Files] MCFG[mcp.json config] EXT[REST Control Extension] end Universal --> KiroLayer style Universal fill:#1a2a3a,stroke:#58a6ff,color:#e6edf3 style KiroLayer fill:#2a1a3a,stroke:#bc8cff,color:#e6edf3

Universal Layer 可搭配任何 AI IDE 使用。只需替換 Kiro-Specific Layer 即可支援 Cursor、Windsurf 等工具。
